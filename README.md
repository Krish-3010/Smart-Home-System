# 🏠 Smart Home Automation System (C++)

A **console-based Smart Home Automation System** built using **Object-Oriented Programming in C++**.  
The application simulates controlling multiple smart home devices through a **menu-driven interface**.

---

## ✨ Features

- 🔌 Control multiple smart devices
- 💡 Manage **Lights**
- 🌬️ Adjust **Fan speed**
- ❄️ Set **Air Conditioner temperature**
- 🔐 Secure **Smart Lock with PIN authentication**
- 📋 Interactive **menu-driven interface**
- 📦 Manage multiple devices using **STL Vector**

---

## 🧠 OOP Concepts Used

- 🧩 Classes and Objects  
- 🧬 Inheritance  
- 🔄 Polymorphism  
- 🔒 Encapsulation  
- 📊 Static Data Members  

---

## ⚙️ Technologies Used

- 💻 **C++**
- 📚 **Standard Template Library (STL)**
- 🧠 **Object-Oriented Programming**

---

## 🚀 How to Run

Compile the program:

```bash
g++ smart_home_system.cpp
```

Run the program:

```bash
./a.out
```
